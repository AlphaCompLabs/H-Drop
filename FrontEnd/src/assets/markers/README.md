# Marcadores do mapa Leaflet

## Arquivos esperados
- `marker-boat.svg` — ícone do barco (posição atual, rotaciona com a proa) — dimensão ideal 32x32
- `marker-boat-shadow.svg` — sombra do barco (opcional)
- `marker-destination.svg` — alvo do waypoint enviado — 24x24
- `marker-home.svg` — ponto de base / origem — 24x24
- `marker-obstacle.svg` — obstáculos detectados (futuro) — 20x20

Se preferir não fornecer SVGs, o app cai em ícones via CSS/`L.divIcon` já definidos por padrão.
