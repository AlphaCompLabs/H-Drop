# Ícones de UI (status bars, menus, alertas)

Recomendado: SVG inline (stroke currentColor) para seguirem o tema automaticamente.

## Status dos subsistemas (barra superior / sidebar)
- `icon-battery.svg` — bateria (SoC) — INA226
- `icon-gps.svg` — status GNSS
- `icon-signal-4g.svg` — força de sinal 4G/LTE
- `icon-mqtt.svg` — conexão com broker MQTT
- `icon-compass.svg` — proa magnética
- `icon-load.svg` — célula de carga HX711 (compartimento ocupado/vazio)
- `icon-obstacle.svg` — detecção de obstáculos
- `icon-motor.svg` — status dos motores brushless
- `icon-speed.svg` — velocidade
- `icon-angular-velocity.svg` — velocidade angular

## Alertas (UI Alerts)
- `icon-critical.svg` — erro crítico (Strategic Red)
- `icon-warning.svg` — aviso (Safety Yellow)
- `icon-info.svg` — informação (Critical Ocean Blue)
- `icon-ok.svg` — sucesso / OK (Life-Saving Green)

## Controles de missão (Sync Test tab)
- `icon-stop.svg` — parar barco
- `icon-home.svg` — retornar à base
- `icon-acknowledge.svg` — confirmar chegada
- `icon-play.svg` — iniciar missão

## UI geral
- `icon-menu.svg` — abrir drawer (mobile)
- `icon-close.svg` — fechar
- `icon-theme-dark.svg` — toggle tema escuro
- `icon-theme-light.svg` — toggle tema claro
